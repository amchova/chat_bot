# chat_bot
Добро пожаловать!
Способы использования:

## Вариант 1: открыть через GitHub Pages

Самый простой способ — перейти по готовой ссылке:
`https://amchova.github.io/chat_bot/`
После открытия страницы *введите команду* `/start` или /help.

## Вариант 2: запустить `index.html` из репозитория GitHub
Нажмите **Code → Download ZIP**.
Откройте папку проекта и запустите файл index.html.
После запуска страницы введите команду /start.
